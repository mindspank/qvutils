Object Extensions

createDivthis.createDiv([divId])
Appends a new DIV-element to the Extension container. The DIV id will the Object Extension id with backslashes replaced by underscores, i.e Document_CH01 or if divId is supplied that will be used as DIV id.

Note: Object Extensions needs to have unique IDs on the page.

Qva.AddExtension('MySampleExtensions', function() {
  var $mydiv = this.createDiv();
  $mydiv.css('backgroundcolor', 'green');
}
